# Healthcare_analysis
Project Overview
This project features an interactive Tableau dashboard designed to monitor and analyze key performance indicators (KPIs) within a healthcare setting. 
The goal is to provide hospital administrators and medical staff with actionable insights into patient demographics, admission trends, and operational efficiency.

🔗 View the Interactive Dashboard on Tableau Public

📊 Key Insights & Features
Executive KPI Ribbon: Real-time tracking of Total Patients, Average Length of Stay (ALOS), and Admission Rates.

Patient Demographics: Deep dive into patient age groups, gender distribution, and insurance types.

Temporal Analysis: Visualization of admission trends over time to identify seasonal peaks and resource requirements.

Departmental Performance: Breakdown of patient volume by medical specialty or department.
